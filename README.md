# 梦想变装 (Dream Dress-Up)

一个让孩子拍照并实现梦想变装的 AI 应用。孩子可以拍照后输入自己的梦想（想成为什么样的人），然后 AI 会生成一张变装后的图片。

## 功能特点

- 拍立得相机风格界面
- 实时摄像头拍照或上传照片
- 上传照片动画效果（照片从底部进入相机）
- 输入姓名（可选）和梦想描述
- AI 根据梦想生成变装图片
- 黑色胶片显影动画效果
- 历史记录保存在浏览器本地
- 软木板风格画廊展示（带图钉效果）
- 照片可在画板上自由拖拽
- 支持下载生成的图片

## 技术栈

- React 18 + TypeScript
- Vite
- CSS3 (响应式设计、动画效果)

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

## API 配置

首次使用时需要配置 API Key。可以通过以下方式配置：

1. 首次生成图片时会弹窗提示输入
2. 通过 URL 参数: `?apiKey=your-api-key`
3. 通过 URL 参数 (JSON): `?settings={"key":"your-api-key","url":"https://api.example.com/v1"}`

API Key 获取地址: https://api.tu-zi.com/token

## 使用说明

1. 打开应用，允许摄像头权限
2. 点击拍照按钮拍摄照片，或点击底部箭头上传本地图片
3. 确认照片后，输入姓名（可选）和梦想描述
4. 点击"确认并生成"，观看黑色胶片从相机顶部弹出
5. 等待 AI 生成完成，观看胶片显影动画
6. 生成的照片会自动出现在画板上，可以自由拖拽
7. 点击右上角"GALLERY"进入软木板风格画廊查看所有记录

## 界面特色

- **拍立得相机**: 复古相机造型，透明镜头显示实时画面
- **胶片动画**: 拍照后黑色胶片缓慢弹出，AI 生成完成后显影
- **画板模式**: 所有照片散落在画板上，可自由拖拽摆放
- **软木板画廊**: 全屏软木板背景，照片用红色图钉固定，随机倾斜

## 注意事项

- 摄像头功能需要在 localhost 或 HTTPS 环境下使用
- 历史记录保存在浏览器 localStorage 中
- 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
