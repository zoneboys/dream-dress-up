# 自定义音效文件

将音频文件放到此目录，文件名对应音效类型：

| 文件名 | 对应音效 |
|--------|----------|
| `shutter.mp3` | 拍照快门 |
| `upload.mp3` | 上传图片 |
| `cameraOn.mp3` | 开启摄像头 |
| `cameraOff.mp3` | 关闭摄像头 |
| `confirm.mp3` | 确认生成 |
| `complete.mp3` | 生成完成 |
| `error.mp3` | 错误提示 |
| `eject.mp3` | 胶片弹出 |
| `developing.mp3` | 显影中（循环播放） |
| `click.mp3` | 点击音 |
| `modeSwitch.mp3` | 模式/模板切换（机械转盘声） |

**支持格式**：`.mp3`、`.wav`、`.ogg`

有自定义文件时优先使用，否则使用默认合成音效。
